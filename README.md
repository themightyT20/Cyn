# AI Voice Transformation and Cloning Platform

A TypeScript-based AI voice transformation and cloning platform with advanced audio processing capabilities, enabling personalized voice experiences through intelligent machine learning techniques.

## Key Technologies

- TypeScript frontend
- Gemini AI integration
- React-based client
- Advanced audio processing
- Machine learning voice cloning
- Multi-modal voice interaction support
- Web Speech API integration for TTS

## Deployment Instructions for Railway

1. Create a new Railway project
2. Connect your GitHub repository to Railway
3. Configure the following environment variables if needed:
   - Any required API keys (for Gemini, OpenAI, etc.)
4. Deploy the application
5. Railway will automatically build and deploy your application using the Procfile

## Local Development

```
npm install
npm run dev
```

## Production Build

```
npm run build
npm run start
```
